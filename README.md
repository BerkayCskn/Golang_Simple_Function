# Golang_Simple_Function
![image](https://user-images.githubusercontent.com/101183663/159083106-ebfd713a-0e84-49c9-a445-9b9d2d1e8e6c.png)

* 'greet' fonksiyonumuz tipi belli olan bir parametre alır
